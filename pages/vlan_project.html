VLAN Based Office Network Project

Networking • VLAN • Packet Tracer

Designed & Implemented by Mahendiran

Project Overview

This project demonstrates the design and implementation of a VLAN-based office network using Cisco Packet Tracer. The goal was to improve network segmentation, security, and performance by logically separating departments using VLANs.

Why This Project Was Created

  • To understand real-time enterprise network structure
  • To implement VLAN segmentation for department-wise isolation
  • To practice router-on-a-stick configuration
  • To improve troubleshooting and networking skills
  • To prepare for Network Engineer / System Admin roles

Network Design & Architecture

  • 1 Router (Inter-VLAN Routing)
  • 1 Managed Layer 2 Switch
  • Multiple PCs & Laptops
  • Separate VLANs for each department

VLAN Configuration Details

VLAN ID Department IP Range Purpose
100 Admin 192.168.10.0/24 Management
200 HR 192.168.20.0/24 Employee Data
300 Marketing 192.168.30.0/24 Business Operations
400 IT 192.168.40.0/24 Infrastructure
500 Guest 192.168.50.0/24 Restricted Access

Real-Time Experience Gained

  • Configured VLANs on managed switches
  • Assigned access & trunk ports correctly
  • Configured Router-on-a-Stick (Sub-Interfaces)
  • Tested inter-VLAN communication
  • Applied security by isolating Guest VLAN
  • Troubleshot IP and connectivity issues
